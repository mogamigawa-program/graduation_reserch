DELETE文の演習（単数条件）

DELETE文は、テーブルから不要なデータを削除するために使用されます。特定の条件に一致する行だけを削除することができます。

例えば、次のようにして employees テーブルから id が 3 の行を削除します。

例:

DELETE FROM employees WHERE id = 3;
