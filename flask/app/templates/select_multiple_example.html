SELECT文（複数条件）実行例

対象テーブル：stationery（文房具）

次のような stationery（文房具） テーブルがあります。

item_idtypecolorprice
1PenBlue120
2EraserRed80
3NotebookGreen250
4PencilYellow100
5PenBlack150

実行例①：種類が"Pen"かつ色が"Blue"

SELECT * FROM stationery WHERE type = 'Pen' AND color = 'Blue';

実行結果： 種類がペンで色が青の文房具のみ表示されます。

item_idtypecolorprice
1PenBlue120

実行例②：種類が"Eraser"または色が"Green"

SELECT * FROM stationery WHERE type = 'Eraser' OR color = 'Green';

実行結果： 種類が消しゴムの文房具全てと、色が緑の文房具が表示されます。

item_idtypecolorprice
2EraserRed80
3NotebookGreen250

実行例③：種類が"Notebook"かつ色が"Green"、または値段が150円以上

SELECT * FROM stationery WHERE (type = 'Notebook' AND color = 'Green') OR price >= 150;

実行結果： 次のいずれかに該当する文房具が表示されます：

  • 種類がノートかつ色が緑
  • 値段が150円以上の文房具
item_idtypecolorprice
1PenBlue120
3NotebookGreen250
5PenBlack150

よくあるエラーと注意点

⚠ 条件に一致する行がない
SELECT * FROM stationery WHERE color = 'Purple' AND type = 'Pen';

→ 実行結果は空になります（該当データが存在しません）。

⚠ 文字列をクォートで囲まない
SELECT * FROM stationery WHERE type = Pen;

→ エラーになります（文字列は 'Pen' のようにシングルクォートで囲みましょう）。

💡 表示カラムの指定
SELECT type, color FROM stationery WHERE price >= 150;

→ * ではなく、必要な列だけを表示することもできます。
