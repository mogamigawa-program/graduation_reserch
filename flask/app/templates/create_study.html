テーブルの作成（CREATE TABLE）

CREATE TABLE 文とは？

テーブルを新しく作るための SQL 文です。 中にどんなデータを入れるのか（名前、データ型、主キーなど）を定義します。

基本構文

CREATE TABLE テーブル名 ( カラム名 データ型, カラム名 データ型, ... );

例：users テーブルを作る

CREATE TABLE users ( id INT PRIMARY KEY, name VARCHAR(50), age INT );

この SQL を実行すると、 id・name・age の3つの項目を持つ users テーブルができます。

ポイント

  • PRIMARY KEY を付けると、そのカラムは重複不可・NULL不可になります
  • VARCHAR(50) は「最大50文字までの文字列」
  • INT は整数を入れるカラム

初心者はまずこのレベルだけ覚えれば十分です。
