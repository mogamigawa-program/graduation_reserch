INSERT文とは

INSERT文は、データベースのテーブルに新しいデータ（レコード）を追加するときに使うSQL文です。
たとえば、新しいユーザー情報や商品データなどを登録したいときに使います。

基本的な書き方

INSERT INTO テーブル名 (列1, 列2, ...) VALUES (値1, 値2, ...);

この構文では、追加したい列と、それに対応する値を指定します。たとえば、「users」テーブルに名前と年齢を追加するには次のように書きます：

INSERT INTO users (name, age) VALUES ('田中太郎', 25);

すべての列に値を入れる場合

すべての列に順番通りに値を入れる場合は、列名の部分を省略することもできます。

INSERT INTO users VALUES (1, '田中太郎', 25);

ただしこの方法は、列の順番や数を間違えるとエラーになるため、できるだけ列名を書くのがおすすめです。

複数行のデータを一度に追加する

複数のレコードを一度に追加することもできます。

INSERT INTO users (name, age) VALUES ('佐藤花子', 30), ('鈴木一郎', 22), ('山田二郎', 28);

注意点

  • NOT NULL 制約がある列には必ず値を入れる必要があります。
  • 文字列は '（シングルクォート） で囲みます。
  • テーブルの構造や制約（PRIMARY KEY, UNIQUE など）に適した値を使いましょう。

まとめ

INSERT文を使えば、テーブルに新しいデータを簡単に追加できます。
まずは基本の形をしっかり覚えて、慣れてきたら応用的な使い方も試してみましょう。
