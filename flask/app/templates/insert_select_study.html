INSERT + SELECT文とは

INSERT + SELECT文は、別のテーブルからデータを取得して、別のテーブルに挿入するためのSQL構文です。
主に「データのコピー」や「一部データの移動」に使われます。

基本構文

INSERT INTO 追加先テーブル (列1, 列2, ...) SELECT 列1, 列2, ... FROM 元テーブル WHERE 条件;

VALUES の代わりに SELECT を使って、複数の行を一括で挿入できます。

使用例

以下のように、年齢が30以上のユーザーを senior_users テーブルに追加できます：

INSERT INTO senior_users (id, name, age) SELECT id, name, age FROM users WHERE age >= 30;

注意点

  • 列数・型が一致している必要があります
  • 挿入先テーブルに存在しない列を指定しないようにしましょう
  • 必要に応じて WHERE で条件をつけましょう
