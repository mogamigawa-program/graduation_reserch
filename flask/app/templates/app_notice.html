このアプリを利用する上での注意事項

1. このアプリで使用している SQL について

本アプリで紹介している SQL 文は MySQL / MariaDB で実際に使用できる書き方に基づいています。 そのため、PostgreSQL・Oracle・SQL Server など他のDBでは構文が異なる場合があります。

2. 実行できる SQL 文について

安全に学習できるよう、実行可能な SQL 文は学習項目に関連するものに制限されています。

  • DELETE, INSERT, UPDATE などの基本構文のみ実行可能
  • DROP DATABASE や ALTER USER など危険な構文は自動でブロック
  • 予想外の操作を避けるため、構文チェックを実施しています

3. ユーザーごとの独立した学習環境

各ユーザーには独立した練習データベースが割り当てられており、 他のユーザーのデータを参照・変更することはできません。

データが壊れても「初期化」ボタンで元に戻せます。

4. 実際の業務データベースとは異なります

本アプリは学習用の簡易環境です。 実務のデータベースでは権限管理、セキュリティ、同時アクセスなど多くの要素が関わります。

5. 実行結果は学習データのみに反映されます

実行結果はあなた専用のサンプルデータだけに反映されます。 個人情報や本番データに影響が及ぶことはありません。

6. エラーは自然なものです

SQL は少しの書き間違いでエラーになりますが、これは正常な学習プロセスです。 エラー文を確認して、少しずつ正しい書き方に近づけていきましょう。

7. 推奨環境

  • PC（スマホでは一部レイアウトが崩れる可能性あり）
  • Chrome / Firefox / Edge の最新バージョン
  • JavaScript 有効

8. 禁止事項

  • ブラウザの「戻る」ボタンを使用してページを遷移しないでください。
  • 不正なSQL文の連続実行
  • 他ユーザーになりすます行為
  • システム改ざん・攻撃目的のアクセス
  • サーバーに高負荷を与える操作
