CREATE TABLE の具体例

例①：ユーザーテーブルの作成

CREATE TABLE users ( id INT PRIMARY KEY, name VARCHAR(50), age INT );

もっとも基本的なテーブルの例です。

例②：商品テーブルの作成

CREATE TABLE items ( id INT PRIMARY KEY, name VARCHAR(100), price INT );

例③：外部キーを使ったテーブル

少しだけ応用ですが、関係データベースっぽくなります。

CREATE TABLE orders ( id INT PRIMARY KEY, user_id INT, item VARCHAR(50), FOREIGN KEY (user_id) REFERENCES users(id) );
