SELECT文（単一条件）実行例

対象テーブル：students

次のような students テーブルがあります。

student_idnameagegrade
1田中18A
2佐藤20B
3鈴木19A

実行例①：年齢が18歳の学生を抽出

SELECT * FROM students WHERE age = 18;

実行結果： 「田中」のデータのみが表示されます。

student_idnameagegrade
1田中18A

実行例②：成績がAの学生を抽出

SELECT name, grade FROM students WHERE grade = 'A';

実行結果： 成績がAの学生（「田中」と「鈴木」）の名前と成績が表示されます。

namegrade
田中A
鈴木A

よくあるエラーと注意点

⚠ 条件に一致する行がない
SELECT * FROM students WHERE age = 25;

→ 実行結果は空になります（該当データが存在しません）。

⚠ 文字列をクォートで囲まない
SELECT * FROM students WHERE name = 佐藤;

→ エラーになります（文字列は '佐藤' のようにシングルクォートで囲みましょう）。

💡 表示カラムの指定
SELECT name, age FROM students WHERE grade = 'A';

→ * ではなく、必要な列だけを表示することもできます。

補足：すべてのデータを確認したい場合

テーブル全体を確認するには、次のように書きます。

SELECT * FROM students;
