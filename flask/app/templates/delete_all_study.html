DELETE文（全レコード削除）

全レコード削除とは？

DELETE文で WHERE 句を省略すると、テーブル内のすべてのデータが削除されます。
テーブル自体は残りますが、中身が空になります。

基本構文

DELETE FROM テーブル名;

WHEREを付けないことで、削除対象を限定せずにすべての行を削除します。

例

例えば、studentsテーブルの全データを削除したい場合：

DELETE FROM students;

これにより、全ての学生データが削除され、テーブルは空の状態になります。

⚠ 注意： この操作は取り消しができません。
実行前にバックアップを取るか、トランザクションを使って安全に実験しましょう。

似た操作との違い

  • DELETE FROM テーブル名; … 中身だけ削除（テーブル構造は残る）
  • TRUNCATE TABLE テーブル名; … 高速に全削除（条件指定不可）
  • DROP TABLE テーブル名; … テーブルそのものを削除
